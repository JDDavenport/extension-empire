# JD Action Items — Extension Empire Launch

## To Ship Extension #1 (QuickCopy Pro)

### 1. Chrome Web Store Developer Account ($5 one-time)
- Go to: https://chrome.google.com/webstore/devconsole
- Register with an anonymous email (e.g., quickcopy.extension@proton.me)
- Pay the $5 developer registration fee

### 2. Generate Extension Icons
- Need 16x16, 48x48, 128x128 PNG icons
- Simple concept: clipboard/copy icon in purple/blue gradient (matches the UI theme)
- Can use any icon generator or ask The Architect to generate via canvas

### 3. Take Screenshots (1280x800)
- Load the extension locally (`chrome://extensions` → Developer Mode → Load Unpacked → select `extensions/quickcopy-pro/`)
- Take 3-5 screenshots showing: popup, context menu, toast notification
- Save to `extensions/quickcopy-pro/listing/`

### 4. Host Privacy Policy
- Options: GitHub Pages, a simple Notion page, or any static URL
- Content is ready in `extensions/quickcopy-pro/listing/privacy-policy.md`

### 5. Submit
- Upload the extension folder to Chrome Web Store
- Use the description from `extensions/quickcopy-pro/listing/description.md`
- Review typically takes 1-3 business days

### Time Required: ~30 minutes total
