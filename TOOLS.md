# TOOLS.md — Extension Empire

## Research
- **web_search** — Chrome Web Store gaps, competitor pricing, niche research
- **web_fetch** — Scrape store listings, read reviews, analyze competitors
- **browser** — Walk through competitor extensions, test UX flows

## Build
- **sessions_spawn** — Spawn builder sub-agents for parallel extension development
- **exec** — Run build scripts, package extensions, test locally
- **write/edit** — Create extension code, manifests, store listings

## Monitor
- **web_fetch** — Check Chrome Web Store stats (if API available)
- **browser** — Check dashboard, review stats
- **cron** — Schedule regular analytics checks

## Distribution
- **browser** — Submit to Chrome Web Store, manage listings
- **web_fetch** — Monitor install counts, review sentiment
