# Extension Pipeline

Extensions queued for development, in priority order.

## Queue

### 1. TabSnap — One-Click Tab Exporter
- **Status:** Next up (after QuickCopy Pro ships)
- **Concept:** Export all open tabs as a list (URLs, titles). One click. Copy to clipboard or download as file.
- **Why:** Universal need, dead simple, many existing options are bloated or broken on MV3
- **Build time:** ~1 day
- **Monetization:** Free: basic export. Pro: scheduled exports, tab groups, markdown format

### 2. JSON Pretty — Clean JSON Formatter
- **Status:** Queued
- **Concept:** Auto-detect and format JSON in browser. Collapsible tree view. Copy paths. 
- **Why:** Developer audience pays well. Search volume high. Existing options dated.
- **Build time:** ~1-2 days
- **Monetization:** Free: format + collapse. Pro: JSONPath queries, diff view, schema validation

### 3. SiteColors — Website Color Palette Extractor
- **Status:** Queued
- **Concept:** Extract full color palette from any page. Show hex/rgb/hsl. Copy colors. Export palette.
- **Why:** Designer/developer crossover niche
- **Build time:** ~2 days

### 4. FormFiller Lite — Custom Form Auto-Fill
- **Status:** Research needed
- **Concept:** Save text snippets for form fields (not passwords). Auto-fill job applications, contact forms.
- **Build time:** ~3 days

### 5. ReadClock — Estimated Reading Time Overlay
- **Status:** Idea stage
- **Concept:** Shows reading time on any article/page. Configurable WPM.
- **Build time:** ~4 hours
