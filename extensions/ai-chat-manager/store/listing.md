## Detailed Description
**The ultimate power tool for ChatGPT and Claude users.**

Tired of losing important conversations? Can't find that one prompt you used last week? AI Chat Manager adds the features that ChatGPT and Claude should have built in.

### 🔍 SEARCH EVERYTHING
Full-text search across ALL your conversations — both ChatGPT and Claude. Find any message instantly.

### 📥 EXPORT CONVERSATIONS
One-click export as Markdown, JSON, or PDF. Never lose an important conversation again.

### 📌 PIN & ORGANIZE
Pin your most important conversations for instant access. Add tags for easy filtering.

### 🗑️ BULK MANAGE
Select and delete multiple old conversations at once. Keep your workspace clean.

### ⚡ PRO FEATURES
- **📁 Smart Folders** — Organize conversations into custom folders
- **🏷️ Color Tags** — Tag and categorize with custom labels
- **⚡ Prompt Library** — Save your best prompts, insert with one click
- **📊 Analytics** — See token usage estimates, topic breakdowns, usage patterns
- **🔗 Share** — Generate clean shareable pages from any conversation

### 🔧 HOW IT WORKS
Install → visit ChatGPT or Claude → conversations are automatically indexed. Use the floating button or popup to search, export, organize, and more.
