# TikTok Shop Seller Tools — Analytics, Auto-Reply & Inventory

## Overview
All-in-one Chrome extension for TikTok Shop sellers. Dashboard overlay, quick reply templates, inventory alerts, order export, and competitor price checking.

## Target Market
TikTok Shop sellers on seller-us.tiktok.com. TikTok Shop grew 153% in 2025, passed Shein/Temu in US sales. Near-zero Chrome extension competition.

## Features
1. **Dashboard Overlay** — Floating panel on seller pages: today's sales/revenue/orders, top 5 products, AOV, trend vs yesterday
2. **Quick Reply Templates** — Pre-saved messages, one-click paste into TikTok Shop message box
3. **Inventory Alerts** — Track stock levels, highlight low-stock in red, browser notifications
4. **Order Export** — CSV export of orders
5. **Competitor Price Checker** — Compare prices vs similar listings

## Monetization
- Free: Dashboard overlay, 3 reply templates, basic analytics
- Pro ($14.99/mo via ExtensionPay): Unlimited templates, CSV export, competitor tracking, inventory alerts, priority support

- Manifest V3, content scripts on seller-us.tiktok.com, dark UI matching TikTok aesthetic
