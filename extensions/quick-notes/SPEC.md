# Quick Notes — Extension Spec

## Overview
Floating sticky note on any webpage, saves per-URL. Clean minimal UI.

## Features
### Free Tier
- One sticky note per page
- Auto-save per URL
- Drag to reposition
- Markdown-lite (bold, italic)

### Pro Tier ($1.99/mo)
- Multiple notes per page
- Color themes (6 colors)
- Export all notes
- Search across notes

## Permissions
- `activeTab` — inject note UI
- `storage` — persist notes per URL
