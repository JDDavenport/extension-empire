# Dark Mode Toggle — Extension Spec

## Overview
One-click dark mode for any website using CSS inversion + smart fixes.

## Features
### Free Tier
- One-click dark mode on any site
- Smart CSS inversion (preserves images)
- Per-site toggle memory
- Keyboard shortcut

### Pro Tier ($1.99/mo)
- Custom brightness/contrast controls
- Site-specific CSS overrides
- Scheduled dark mode (sunset/sunrise)
- Whitelist/blacklist sites

## Permissions
- `activeTab` — inject dark mode CSS
- `storage` — remember per-site preferences
