# Reading Time — Extension Spec

## Overview
Shows estimated reading time on any article/blog. Injects small badge.

## Features
### Free Tier
- Auto-detect article content
- Reading time badge (top of page)
- Word count display
- Works on any page with text content

### Pro Tier ($0.99/mo)
- Reading speed customization (WPM)
- Progress bar while reading
- Time remaining display
- Reading stats dashboard

## Permissions
- `activeTab` — read page content for word count
