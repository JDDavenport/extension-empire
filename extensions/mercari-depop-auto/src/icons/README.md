# RelistPro Icons

Generate these icon files:

- `icon16.png` — 16×16px — Toolbar favicon
- `icon48.png` — 48×48px — Extensions page
- `icon128.png` — 128×128px — Chrome Web Store / install dialog

**Design spec:**
- Purple lightning bolt (⚡) on a dark rounded-square background
- Background: linear gradient from #6C5CE7 to #a855f7
- Icon: white lightning bolt, centered
- Corner radius: ~20% of size
- Clean, minimal, modern — should look sharp at 16px
