# Color Picker Pro — Extension Spec

## Overview
Eyedropper tool to pick colors from any webpage, saves palette history, copies hex/rgb/hsl.

## Features
### Free Tier
- Pick any color from webpage
- Copy as HEX, RGB, HSL
- Save last 20 colors
- One-click copy to clipboard

### Pro Tier ($2.49/mo)
- Unlimited palette history
- Named palettes / collections
- Export palettes (CSS, JSON, ASE)
- Color harmony suggestions

## Permissions
- `activeTab` — pick colors from page
- `storage` — save palette history
